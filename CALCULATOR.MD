CALCULATOR
=========

##Commands
I/O|Function|Notes
---|----|----
First Input|Operation|1=+, 2=-, 3=*, 4=/,(Invalid input closes program)
Second Input|First Number|
Third Input|Second Number|Number to be subtracted/divided from base
Output|Answer|-1 for division is error code

##Features
- [ ] Multiply
- [ ] Add
- [ ] Subtract
- [ ] Divide
 - [ ] Remainder
 - [ ] Decimals?
- [ ] Indecies
- [ ] Root
