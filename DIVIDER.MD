DIVIDER
=======

This program divides numbers

###CODE
```
DIV LDA NULL
INP
STA FIRST
INP
STA SECOND
LDA NULL
STA COUNT
DIVLOOP LDA FIRST
SUB SECOND
STA FIRST
LDA COUNT
ADD ONE
STA COUNT
LDA FIRST
BRZ END
BRP DIVLOOP
BRA ERR
HLT
END LDA COUNT
OUT
HLT
ERR LDA NEGATIVE
OUT
HLT
ONE DAT 1
FIRST DAT
SECOND DAT
COUNT DAT
NULL DAT 0
NEGATIVE DAT -1
```
